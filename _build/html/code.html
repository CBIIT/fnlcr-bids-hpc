Documentation for the image_segmentation module.

image_segmentation.image_augmentation.augment_images(images, masks=None, num_aug=1, do_composite=True, output_dir=None, composite_sequence=None, individual_seqs_and_outnames=None, aug_params=None)¶

Augment images and/or masks.

Parameters

  • images – Images to augment; NumPy array of shape (H,W), (H,W,3), (N,H,W), or (N,H,W,3); values can be in range [0,1], [0,2^8-1], or [0,2^16-1]

  • masks – (Optional) Masks to correspondingly augment; NumPy array of shape (H,W) or (N,H,W); values are 0 and positive integers

  • num_aug (int.) – (Optional) Number of augmentations to perform; number of output images will be N * num_aug

  • do_composite (bool.) – (Optional) Whether to do composite augmentations (multiple augmentations at once) or individual augmentations (for observing the effect of each augmentation); only has an effect if either (1) both composite_sequence and individual_seqs_and_outnames are None or (2) both composite_sequence and individual_seqs_and_outnames are set

  • output_dir (str.) – (Optional) If not set to None, location where .tif images should be saved for observation purposes; if set to None, no saving will be done

  • composite_sequence (func.) – (Optional) Function specifying the sequence of augmentations to perform all at once; if set to None, the default example_composite_sequence (below) is used; to customize the composite sequence, create a function taking example_composite_sequence as an example

  • individual_seqs_and_outnames (func.) – (Optional) Function specifying the individual augmentations to perform; if set to None, the default example_individual_seqs_and_outnames (below) is used; to customize the individual augmentations, create a function taking example_individual_seqs_and_outnames as an example

  • aug_params (cls.) – (Optional) Class specifying the augmentation parameters to apply to either composite or individual augmentations; if set to None, the default example_augmentation_parameters (below) is used; to customize the augmentation parameters, create a class taking example_augmentation_parameters as an example

Returns

  • If do_composite=True: augmented images ((N,H,W,C)), and, if masks were input, augmented masks ((N,H,W)); these are both NumPy arrays of dtype=’uint8’

  • If do_composite=False: list of augmented images ((N,H,W,C)), one for each individual augmentation; these are all NumPy arrays of dtype=’uint8’

As mentioned above, note that do_composite can be set by setting one (and only one) of composite_sequence and individual_seqs_and_outnames to functions (i.e., not None)
